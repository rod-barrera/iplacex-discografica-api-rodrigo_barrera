# API REST Spring para administración de discografías favoritas de empleados de IPLACEX

API REST desarrollada con Spring Boot para administrar artistas y discos favoritos, utilizando MongoDB Atlas como base de datos.

## Tecnologías

- Java 21
- Spring Boot
- Spring Web
- Spring Data MongoDB
- MongoDB Atlas
- Gradle
- Docker
- Render

## Base de datos

Base de datos: `discografica-db`

Colecciones:

- `artistas`
- `discos`

## Variables de entorno

- `MONGODB_URI`
- `MONGODB_DATABASE`
- `PORT`

## Despliegue

Proyecto preparado para desplegarse en Render mediante Docker.
